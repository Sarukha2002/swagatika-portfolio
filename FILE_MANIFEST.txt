🔐 SECURITY VERIFICATION
═════════════════════════════════════════════════════════════════════════════

After deployment, verify security headers:
1. Open DevTools (F12)
2. Go to Network tab
3. Click on any request to your site
4. Check Response Headers for:

Should see:
✅ X-Frame-Options: SAMEORIGIN
✅ X-Content-Type-Options: nosniff
✅ X-XSS-Protection: 1; mode=block
✅ Referrer-Policy: strict-origin-when-cross-origin
✅ Strict-Transport-Security: max-age=31536000

If missing:
→ Verify vercel.json or _headers deployed correctly
→ Hard refresh browser (Ctrl+Shift+R)


📞 SUPPORT & TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

Common Issues:
─────────────

Q: "Domain not resolving"
A: 1. Check DNS settings in registrar
   2. Wait 24-48 hours for propagation
   3. Test: nslookup your-domain.com

Q: "/resume redirect not working"
A: 1. Verify resume.pdf exists at /public/assets/resume.pdf
   2. Check vercel.json or _redirects deployed
   3. Hard refresh browser (Ctrl+Shift+R)

Q: "Analytics not tracking"
A: 1. Open Console (F12)
   2. Should see: "[Telemetry] Portfolio analytics initialized"
   3. Check Plausible script loaded (Network tab, search "plausible")
   4. Click button and verify event logs to console

Q: "PageSpeed score below 90"
A: 1. Run https://pagespeed.web.dev on your domain
   2. Check "Opportunities" section for top issues
   3. Common: Missing image dimensions, render-blocking scripts
   4. See PERFORMANCE_OPTIMIZATION.md for fixes

Q: "Favicon not showing"
A: 1. Hard refresh (Ctrl+Shift+R or Cmd+Shift+R)
   2. Check favicon link in HTML <head>
   3. Verify data:image/svg+xml syntax is correct
